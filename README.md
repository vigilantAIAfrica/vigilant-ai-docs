## 🛰️ Week 3 Satellite Tools (Phase 1)
This week, we deployed standalone "Satellite" apps to tackle specific fraud challenges:
- **Swahili Scam Translator:** Bridging the language gap for 200M+ speakers using Helsinki-NLP.
- **AI Mobile Scam Spotter:** Leveraging DistilBERT for real-time sentiment analysis of aggressive phishing.
- **Basic Input Checker:** A FastAPI-based security shield to prevent SQL injection and bad data hacks.
- **Process Monitor:** An offline heartbeat tool to ensure system stability in rural/low-spec environments.

## 🛠️ Quick Start
1. **Clone the repo:** `git clone https://github.com`
2. **Install MkDocs:** `pip install mkdocs`
3. **Serve locally:** `mkdocs serve`
4. **Browse:** `http://localhost:8000`
