## Features
- Beginner-friendly guides for Kenyan developers.
- SDG-aligned project overviews.
- API and usage docs for easy integration.

## Tech Stack
- Markdown for docs.
- MkDocs or GitHub Wiki for rendering (free).
- Diagrams.net for visuals.

## Quick Start
1. Clone the repo: `git clone https://github.com/VigilantAI/vigilant-ai-docs.git`
2. Install MkDocs: `pip install mkdocs`.
3. Serve locally: `mkdocs serve`.
4. Browse at http://localhost:8000.
